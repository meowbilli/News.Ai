📌 News.Ai

News.Ai is a modern Flask-based News platform developed by Furkan Khan. It allows users to create and share news while integrating real-time fact-checking using Google's Gemini AI.

💡 Key Features

  • User registration, login, and profile management
  • Create, edit, and delete posts
  • Like and report functionality for each post
  • AI-based fake news detection using Gemini
  • Gemini response labels:
    • ✅ News is verified by Gemini
    • ⚠️ Gemini suggested this as fake
  • Automatic warning if reports exceed likes
  • Responsive and clean user interface

🔍 How Gemini Works in BlogApp

When a post is created, its content is automatically sent to the Gemini API with a custom prompt asking whether the information is likely true or fake. Based on the AI’s response, a label is displayed on the post to inform readers about the reliability of the content.
